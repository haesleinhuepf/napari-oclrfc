## Installation

You can install `napari-oclrfc` via [pip]:

    git clone https://github.com/haesleinhuepf/oclrfc
    cd oclrfc
    pip install -e .
    git clone https://github.com/haesleinhuepf/napari-oclrfc
    cd napari-oclrfc
    pip install -e .
